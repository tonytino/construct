# ADR Template — not a decision

> This file is the starting template for new ADRs. It is **not** a decision record itself. The underscore prefix keeps it separate from numbered ADRs. When authoring a new ADR, copy this file to `NNN-short-slug.md` where `NNN` is the next unused three-digit number.

## Status

[Accepted | Superseded by ADR-NNN | Deprecated]

## Context

What problem or decision point existed? Why did it need to be resolved?

## Decision

What we chose and why. Keep this to 2-3 sentences.

## Consequences

What this means in practice for agents working in the codebase. Focus on concrete implications: what to do, what not to do, what changes as a result.
